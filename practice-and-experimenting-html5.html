'Audio' Element

The 'audio' element is part of the 'embedded' content model in HTML5 and allows audio files to be embedded into and controlled from a webpage.

'Video' Element

The 'video' element is also part of the 'embedded' HTML5 content model and allows videos to be embedded in a webpage.

Aside content is secondary content that is separate from the main article/section content but still indirectly related.

This is the 'Footer' element
